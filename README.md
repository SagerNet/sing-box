### 安装

sing-box 需要 Golang 1.18.5 或更高版本
### 安装 Golang
```jaon
cd
curl -fsL https://raw.githubusercontent.com/jetsung/golang-install/main/install.sh | bash
source /root/.bashrc
```
### 安装最新的 sing-box 版本
```
go install -v github.com/sagernet/sing-box/cmd/sing-box@latest
````
