# SFI

实验性的 iOS sing-box 客户端。

#### 要求

* iOS 15.0+
* macOS 12.0+ with Apple Silicon

#### 下载

* [TestFlight](https://testflight.apple.com/join/c6ylui2j)

#### 注意事项

* 远程配置文件请求中的 User Agent 为 `SFI/$version ($version_code; sing-box $sing_box_version)`
* 崩溃日志位于 `设置` -> `查看服务日志`

#### 隐私政策

* SFI 不收集或共享个人数据。
* 软件生成的数据始终在您的设备上。
