# Fake IP

### Structure

```json
{
  "dns": {
    "servers": {
      "type": "fakeip",
      "tag": "",

      "inet4_range": "198.18.0.0/15",
      "inet6_range": "fc00::/18"
    }
  }
}
```

### Fields

#### inet4_range

IPv4 address range for FakeIP.

#### inet6_address

IPv6 address range for FakeIP.
