If enabled in the inbound, the protocol and domain name (if present) of by the connection can be sniffed.

#### Supported Protocols

|  Network  |  Protocol  |  Domain Name  |
|:---------:|:----------:|:-------------:|
|    TCP    |    HTTP    |     Host      |
|    TCP    |    TLS     |  Server Name  |
|    UDP    |    QUIC    |  Server Name  |
|    UDP    |    STUN    |       /       |
