# Service

### Structure

```json
{
  "services": [
    {
      "type": "",
      "tag": ""
    }
  ]
}
```

### Fields

| Type       | Format                 |
|------------|------------------------|
| `ccm`      | [CCM](./ccm)           |
| `derp`     | [DERP](./derp)         |
| `resolved` | [Resolved](./resolved) |
| `ssm-api`  | [SSM API](./ssm-api)   |

#### tag

The tag of the endpoint.
