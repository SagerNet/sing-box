# 服务

### 结构

```json
{
  "services": [
    {
      "type": "",
      "tag": ""
    }
  ]
}
```

### 字段

| 类型       | 格式                   |
|-----------|------------------------|
| `ccm`     | [CCM](./ccm)           |
| `derp`    | [DERP](./derp)         |
| `resolved`| [Resolved](./resolved) |
| `ssm-api` | [SSM API](./ssm-api)   |
