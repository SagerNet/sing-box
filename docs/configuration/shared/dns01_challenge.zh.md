### 结构

```json
{
  "provider": "",
  
  ... // 提供商字段
}
```

### 提供商字段

#### Alibaba Cloud DNS

```json
{
  "provider": "alidns",
  "access_key_id": "",
  "access_key_secret": "",
  "region_id": ""
}
```

#### Cloudflare

```json
{
  "provider": "cloudflare",
  "api_token": ""
}
```

#### Tencent Cloud

```json
{
  "provider": "tencentcloud",
  "secret_id": "",
  "secert_key": ""
}
```
